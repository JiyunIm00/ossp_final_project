# Project - SKKU Corona Map

## 1. A brief overview

-   SKKU에서는 코로나19 확진자가 발생하면 학생들에게 메세지를 발송하고 '[코로나-19 종합안내 홈페이지](https://skb.skku.edu/haksaeng/index.do)'에 관련 정보를 업데이트 한다. 하지만 텍스트로만 공지되는 특성상 교내 확진자 동선을 단번에 파악하기 어렵다. 따라서 이를 **시각적으로 표현**하려 한다.
